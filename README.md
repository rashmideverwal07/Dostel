# Dostel 

Dostel is an Android application designed to help students discover and connect with hostels and fellow students.  
The project was built as a learning-based application to understand Android development and Firebase integration.

---

##  Features
- User authentication using Firebase
- Storage and retrieval of student and hostel data
- Image upload and display using Firebase Storage
- Scrollable hostel listings
- Clean and simple user interface

---

##  Tech Stack
- **Language:** Kotlin  
- **IDE:** Android Studio  
- **UI:** Jetpack Compose (Material 3) and XML-based layouts  
- **Backend:** Firebase  
  - Firebase Authentication  
  - Firebase Realtime Database  
  - Firebase Firestore  
  - Firebase Storage  
- **Libraries:** RecyclerView, Glide  

---

##  Architecture
- Activity-based Android architecture  
- Intent-based navigation between screens  
- Hybrid UI approach using Jetpack Compose and XML  

---

##  Testing
- Tested on a physical Android device  
- APK generated and installed successfully  

---

##  Project Status
This project is currently not deployed on the Play Store and was developed for learning and academic purposes.
